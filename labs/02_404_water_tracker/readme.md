# 🌊 404 Water Quality Tracker

## 💡 Why I Built This
I thought it would be a good idea to build an **iOS app** for tracking my betta fish's water quality. 

## 🛠️ What It Does
This will track three main things:
1. **Water Quality:** pH, Ammonia, Nitrate, Nitrite (with trend visualization)
2. **Feeding:** Track up to four different food types with quantity logging
3. **Activity:** Daily activity level tracking for my fish

## 🏗️ Architecture & Phases
### Phase 1 — Prompting & Planning
This is a new approach—I'm treating this as an **AI-Assisted Learning** project. I define the requirements and architecture, let GPT generate the modules, and I perform the **debugging and refactoring**. This helps me understand the "how" and "why" behind the code structure.

### Phase 2 — Debugging & Refactoring
The current phase where I learn the logic, fix the inevitable AI bugs, and mold the code into a scalable architecture.

## 💻 Technical Stack
